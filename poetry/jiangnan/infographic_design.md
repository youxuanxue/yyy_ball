# 《江南》信息图设计方案

## 📐 设计概述

- **目标受众**：6-10岁小学生
- **设计风格**：手绘风格、卡通可爱
- **画面尺寸**：建议 1080×1920 像素（竖版）或 1920×1080 像素（横版）
- **主色调**：清新绿色系 + 粉色荷花 + 蓝色水波

---

## 🎨 视觉风格指南

### 整体风格
- 手绘插画风格，线条柔和
- 使用水彩/蜡笔质感
- 卡通化的人物和动物形象
- 圆润的边框和元素

### 配色方案
| 用途 | 颜色 | 色值参考 |
|------|------|----------|
| 莲叶 | 翠绿色 | #4CAF50, #81C784 |
| 荷花 | 粉红色 | #F48FB1, #F06292 |
| 水面 | 淡蓝色 | #81D4FA, #4FC3F7 |
| 鱼儿 | 金橙色 | #FFB74D, #FF8A65 |
| 背景 | 米黄色 | #FFF8E1, #FFFDE7 |
| 文字 | 深棕色 | #5D4037 |

### 字体建议
- 标题：圆润可爱的手写字体（如：站酷快乐体、萌趣果冻体）
- 正文：清晰易读的卡通字体（如：华康少女字体、方正卡通简体）

## 🖼️ 插图元素清单

### 核心元素（必须包含）
1. **莲叶群** - 翠绿色的荷叶，密密层层
2. **荷花** - 粉色/白色荷花，含苞待放或盛开
3. **小鱼** - 金色/红色小鱼，表情活泼可爱
4. **水波纹** - 浅蓝色的水面波纹
5. **采莲人** - 卡通小女孩坐在小船上

### 装饰元素
- 蜻蜓、蝴蝶
- 阳光、云朵
- 远山、柳树
- 水珠、涟漪

### 图标元素
- 📚 书本图标 - 表示学习/知识
- 🏛️ 古建筑图标 - 表示历史/古代
- 🎵 音符图标 - 表示歌谣/音乐
- 💡 灯泡图标 - 表示要点/小知识

## 📝 文案内容

### 标题区
```
🌸 《江南》古诗趣味学 🌸
汉乐府民歌 | 适合小学生
```

### 古诗原文区
```
江南可采莲，莲叶何田田。
鱼戏莲叶间。
鱼戏莲叶东，鱼戏莲叶西，
鱼戏莲叶南，鱼戏莲叶北。
```

### 作者来源区
```
📚 什么是"汉乐府"？

🏛️ 这是2000多年前皇帝设立的音乐机构
🎵 专门派人收集民间的歌谣
👨‍👩‍👧‍👦 《江南》就是老百姓集体创作的哦！
```

### 创作背景区
```
🏞️ 古诗的故事背景

📍 江南 = 长江以南的美丽水乡
🌿 那里河流湖泊多，到处是莲花
☀️ 每年夏天，人们划船采莲
🎶 大家边采莲边唱歌，好快乐！
```

### 诗歌含义区
```
📖 古诗是什么意思？

🌿 田田 = 莲叶长得茂盛整齐的样子
🐟 戏 = 开心地嬉戏玩耍
🔄 东西南北都提到 = 小鱼游遍了每个角落！
```

### 意义价值区
```
✨ 这首诗教会我们

🌈 感受大自然的美丽
😊 体会劳动的快乐
🤝 人与自然和谐相处
📜 了解古人的生活
```

### 趣味小知识区
```
🎯 你知道吗？

莲花全身都是宝！
🌸 荷花 → 观赏     🫛 莲子 → 营养美食
🥢 莲藕 → 清甜可口  🍃 荷叶 → 包粽子
```

## 🛠️ 制作工具推荐

### 设计软件
- **Canva** - 简单易用，有丰富模板
- **Figma** - 专业设计，协作方便
- **Procreate** (iPad) - 手绘效果最佳
- **Adobe Illustrator** - 矢量图形专业工具

### 素材资源
- **Undraw** - 免费插画素材
- **Flaticon** - 图标素材库
- **Freepik** - 矢量素材
- **千图网/摄图网** - 国内素材平台

### AI 辅助工具
- **Midjourney** - 生成手绘风格插图
- **DALL-E** - AI图像生成
- **Stable Diffusion** - 开源图像生成
