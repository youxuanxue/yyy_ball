# 孙子兵法第四课：军形篇 - 做个不倒翁

## 1. 封面
**标题**：怎么做“常胜将军”？
**副标题**：孙子兵法第四课：军形篇
**口播**：大家好！欢迎回到《我是小小谋略家》。今天是第四课。大家都想当赢家，但你知道怎么做才能“永远不输”吗？孙武爷爷有一个超级秘密！

## 2. 也是一种胜利（先为不可胜）
**核心概念**：善战者，先为不可胜，以待敌之可胜。
**画面**：坚固的城堡/穿盔甲的小人 vs 在等待机会。
**口播**：孙武爷爷说：“先为不可胜”。意思是，打仗前，先把自己练得棒棒的，像穿了钢铁侠的盔甲一样，谁也打不倒你。只要我不输，我就已经立于不败之地啦！

## 3. 赢的秘密（在己 vs 在敌）
**核心概念**：不可胜在己，可胜在敌。
**画面**：左边（自己控制）：努力复习、锻炼身体；右边（别人控制）：对手生病、对手犯错。
**口播**：记住这句话：“不可胜在己，可胜在敌。” 意思是：我不被你打败，这事儿归我管；但我能不能打赢你，那得看你犯不犯错。就像考试，我能保证自己复习好，但不能保证考卷出得简不简单。

## 4. 赢家和输家的区别（胜兵先胜）
**核心概念**：胜兵先胜而后求战，败兵先战而后求胜。
**画面**：
*   赢家：准备充分（满血满装备）-> 去打仗 -> 轻松赢。
*   输家：不管三七二十一（没装备）-> 去打仗 -> 盼着赢。
**口播**：聪明的将军，都是先在家里算好了肯定能赢，才出门去打仗；而笨蛋将军，是什么都没准备好，冲上去跟人家打，心里祈祷“老天保佑让我赢吧”。你是哪一种呢？

## 5. 结尾互动
**口播**：所以，真正的胜利，是在打仗之前就决定的！
